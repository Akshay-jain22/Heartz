## Setup Project

### Install dependencies

-   cd client
-   npm install

### Run Server

-   Open Terminal
-   python server.py

### Run Client

-   Open Terminal
-   cd client
-   npm start

Runs the app in the development mode.\
Open [http://localhost:3000](http://localhost:3000) to view it in your browser.
